6.2 Employee Salary Records

Scenario: An organisation stores employee records in salary.txt in the format Employee ID, Name, Salary. Write a program to update the salary of a given Employee ID and save the changes back to the file.

Algorithm:

1. Input employee ID and new salary.

2. Open salary.txt in read mode and read all lines.

3. If employee ID matches, update salary.

4. Write all lines back to the salary file.

5. Display confirmation message.

6. Sample File: salary.txt

101, Arun, 50000

102, Bharya, 60000 

103, Chitra, 55000 

104, Deepak, 45000

output:

Enter employee ID to update: 103

Enter new salary: 58000

Salary updated successfully.
